Why Do We Split the Data?

If we train and test on the same data, the model just memorises the answers — like giving a student the exam paper in advance. The accuracy would be falsely high.

By keeping a separate test set that the model has never seen, we get a realistic accuracy score.

Note: The original project scored accuracy on the training set and reported 99.16%. The correct test-set accuracy is 98.71%.
