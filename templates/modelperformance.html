Algorithm Descriptions

Random Forest — Builds 100 decision trees on random subsets of data and takes a majority vote. Reduces overfitting. Best performer.

Bagging — Bootstrap aggregation. Similar to Random Forest but without feature randomisation.

Gradient Boosting — Builds trees sequentially, each one fixing the errors of the previous.

AdaBoost — Adaptive boosting. Focuses on misclassified samples each round.

Logistic Regression — Simple linear model. Fast but weakest for non-linear traffic patterns.
