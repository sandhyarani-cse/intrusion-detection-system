About the Dataset

NSL-KDD is the standard benchmark dataset for network intrusion detection research. It contains 38 features per connection record.

Label 0 = Normal Traffic  ·  Label 1 = Attack

Four attack types are present: DoS (Denial of Service), Probe (scanning), R2L (remote to local), and U2R (privilege escalation).
